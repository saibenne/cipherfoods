# CipherFoods Customer Web App

> Next.js 14+ customer-facing storefront — Phase 1 deliverable

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS + shadcn/ui
- **State**: Zustand + TanStack Query
- **Auth**: Keycloak via next-auth

## Features (Planned)
- Product browsing with full-text search
- Category navigation
- Product detail pages with variants
- Shopping cart (multi-vendor)
- Checkout flow with Razorpay
- Order tracking
- User profile and order history
- Responsive design (mobile-first)

## Getting Started
```bash
# From workspace root
npm run start:web
```
