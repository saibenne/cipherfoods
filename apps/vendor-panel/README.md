# CipherFoods Vendor Panel

> Next.js 14+ vendor/farmer management dashboard — Phase 2 deliverable

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS + shadcn/ui
- **State**: Zustand + TanStack Query
- **Auth**: Keycloak via next-auth (vendor roles)

## Features (Planned)
- Vendor onboarding with KYC upload
- Product management (CRUD, variants, pricing)
- Order management and fulfillment
- Inventory tracking
- Earnings dashboard and payout history
- Store settings and profile

> This app will be scaffolded in Phase 2 of development.
