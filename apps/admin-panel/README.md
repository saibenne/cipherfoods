# CipherFoods Admin Panel

> Next.js 14+ platform administration dashboard — Phase 2 deliverable

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS + shadcn/ui
- **State**: Zustand + TanStack Query
- **Auth**: Keycloak via next-auth (admin roles)

## Features (Planned)
- Platform dashboard (orders, revenue, users)
- Vendor management (KYC approval, suspensions)
- User management
- Order oversight
- Category and commission management
- Promotion and coupon management
- Support ticket overview
- Platform configuration

> This app will be scaffolded in Phase 2 of development.
